# [WIP] WebVR SkyMap

DEMO: https://binzume.github.io/webvr-skymap/


WebVRで星空を表示するやつです

- WebVR対応ブラウザ + コントローラが必要です (Oculus Go/Questで動作確認)
- 星のデータはヒッパルコス星表を利用しています
- 星座線は http://astronomy.webcrow.jp/hip/
- 太陽と月が表示されていますが，おおよその方角です(日食などは再現されません)
